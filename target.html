Target for CSZNET

Usage Instructions

If you need to perform a 301 redirect from a.com to b.com:

Simply point a.com to t.csz.net or target.csz.net using a CNAME record.

Then, add a TXT record with target.a.com and its value set to https://b.com.

If you want to preserve URL parameters during the redirection, set the value to https://b.com/*.
